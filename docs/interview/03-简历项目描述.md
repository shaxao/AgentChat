# 简历项目描述

## 项目名称

AutoCode AI 对话与代码开发平台

## 项目简介

基于 Spring Boot、FastAPI、React、MySQL、WebSocket/SSE 构建的 AI Agent 平台，支持普通 AI 对话、多模型渠道管理、用户记忆、技能市场、RBAC 权限、统一计费，以及面向代码开发的 AutoCode Agent。AutoCode 可以根据用户需求自主读取项目、修改代码、运行验证、分析失败并继续修复，同时支持云端工作区和本地 Runner 执行。

## 技术栈

- 前端：React、TypeScript、Vite、Tailwind CSS、lucide-react、SSE、WebSocket
- Java 后端：Spring Boot、MyBatis-Plus、MySQL、JWT、RBAC、SSE、钱包与订阅计费
- AutoCode 后端：FastAPI、Python asyncio、WebSocket、Agentic Loop、Tool Registry、SystemContext、CacheLedger
- 工程能力：Nginx、systemd、PowerShell 部署脚本、MySQL 迁移、日志治理、本地 Runner

## 我的职责

- 设计并实现 AutoCode 核心 Agentic Loop，将固定阶段流水线升级为自主决策执行模式。
- 设计 SystemContext Epoch，上下文源通过 manifest、hash、epoch、summary 管理，减少重复读取和 prompt 膨胀。
- 设计平台级 CacheLedger 缓存账本，支持 AutoCode 和 Java 对话系统复用缓存统计、命中率、token 节省估算。
- 实现 Tool Registry、Tool Output Store 和权限审批机制，统一工具元数据、风险等级、中文展示和大输出收敛。
- 设计本地 Runner，通过 WebSocket 让用户本地项目可被 Agent 安全读写和验证，降低服务器资源消耗。
- 接入统一场景化计费策略，支持 chat/autocode/workflow 独立额度、钱包兜底和 Coding Plan 上游成本模式。
- 优化任务继续修改、运行中插话、完成任务再次开发、CI 失败自动修复等 Agent 任务生命周期问题。
