# AutoCode 面试材料索引

这组文档是从完整架构复盘稿拆出来的面试准备材料，适合不同场景直接使用。

## 推荐阅读顺序

1. [01-三分钟项目介绍.md](01-三分钟项目介绍.md)
   - 用于面试开场，快速讲清楚项目是什么、解决什么问题、你的技术贡献是什么。

2. [02-十分钟架构深度讲解.md](02-十分钟架构深度讲解.md)
   - 用于架构追问，重点讲 Agentic Loop、上下文、缓存、权限、本地 Runner、计费。

3. [03-简历项目描述.md](03-简历项目描述.md)
   - 可直接改写进简历，包含项目描述、技术栈、职责、亮点、量化表达。

4. [04-高频面试问答.md](04-高频面试问答.md)
   - 面试官常问问题的回答模板，偏“怎么讲得专业”。

5. [05-架构图与流程图.md](05-架构图与流程图.md)
   - Mermaid 图，包括整体架构、Agentic Loop、缓存分层、本地 Runner、计费流程。

6. [06-问题复盘与优化清单.md](06-问题复盘与优化清单.md)
   - 讲开发过程中遇到的问题、定位方式、解决方案和后续优化方向。

## 一句话定位

AutoCode 是一个面向 AI 对话和代码开发的 Agent 平台：Java 主系统承载用户、对话、模型、权限、记忆、技能和计费；Python AutoCode 子系统承载 Agentic Loop、工具执行、工作区、本地 Runner、上下文和缓存治理。
